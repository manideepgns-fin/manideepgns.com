# manideepgns.com

Personal site — [Manideep GNS](https://manideepgns.com).

Finance × Curiosity × Code. Single-page portfolio, no framework, no build step.

## Stack

- One `index.html`. One photo. That's it.
- System font stack — no webfont request.
- All motion is gated on `prefers-reduced-motion`.
- Hosted on Cloudflare Pages, auto-deployed from `main`.

## Local

Open `index.html` in a browser. Or:

```
python -m http.server 8080
```
